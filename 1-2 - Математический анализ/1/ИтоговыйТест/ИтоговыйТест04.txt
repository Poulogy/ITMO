Функция f:E->R непрерывна в точке х0, если для любой V(f(x0)) существует U(x0): для любого x0 принадлежащего пересечению U(x0) и E, f принадлежит V(f(x0))

Функция f:E->R непрерывна в точке х0, если для любого e>0 ∃δ = δ(e)>0: для любого x принадлежащего E: |x-x0|<δ |f(x)-f(x0)|<e.

Пусть f : E → R. Если x0 ∈ R – предельная для E и f не непрерывна в точке
x0, то точка x0 называется точкой разрыва для функции f.

Понятие устранимого разрыва
Пусть f : E → R и x0 ∈ R. Если существует limx→x0
f(x) = A ∈ R, но значение
функции в точке x0 либо не определено, либо отличается от A, то x0 называется
точкой устранимого разрыва функции f.

Понятие разрыва 1-ого рода (скачка)
Пусть f : E → R. Если существуют односторонние пределы f(x0 ± 0) ∈ R, но
f(x0 + 0) ̸= f(x0 − 0),
то точка x0 называется точкой разрыва первого рода или скачком.

Понятие разрыва 2-ого рода
Пусть f : E → R, x0 – предельная для E. Если не существует хотя бы одного из
односторонних пределов f(x0 ± 0) в R, то точка x0 называется точкой разрыва
второго рода.


(Первая теорема Больцано–Коши).
Пусть f ∈ C[a, b] и f(a) · f(b) < 0. Тогда
∃c ∈ (a, b) : f(c) = 0.

Доказательство. Пусть, для определенности, f(a) > 0, f(b) < 0. Обозначим a_1 = a,
b_1 = b. Разделим отрезок I_1 = [a_1, b_1] пополам точкой
c_1 =(a_1 + b_1)/2
Если f(c1) = 0, то доказательство закончено. Если f(c_1) ̸= 0, то либо f(c1) > 0, либо
f(c1) < 0. Из получившихся двух отрезков выберем тот, на концах которого значения
функции все также имеют разный знак. Это значит, что в первом случае в качестве
отрезка I_2 выберем отрезок [c_1, b], а во втором случае в качестве отрезка I2 выберем
отрезок [a, c_1]. В любом из двух случаев концы нового отрезка обозначим a2 и b2.
Продолжаем по индукции. Если построен отрезок I_n−1 = [a_n−1, b_n−1], то на шаге
n ≥ 2 разделим отрезок In−1 пополам точкой
c_n−1 = (a_n−1 + b_n−1)/2
Если f(c_n−1) = 0, то доказательство закончено. Если f(c_n−1) ̸= 0, то либо f(c_n−1) > 0,
либо f(c_n−1) < 0. В первом случае в качестве отрезка I_n выберем отрезок [c_n−1, b_n−1], а 
во втором случае в качестве отрезка I_2 выберем отрезок [a_n−1, c_n−1]. В любом из
двух случаев концы нового отрезка обозначим a_n и b_n.
Так как an, bn ∈ [a, b], то по теореме Больцано–Вейерштрасса
∃a_n_k: a_n_k −→(k→∞)a_0, 
∃b_n_k: b_n_k −→(k→∞)b_0, причем a0, b0 ∈ [a, b].
Так как длина отрезка I1 на каждой итерации уменьшается в два раза, то
bn − an = (b1 − a1)/2^(n−1)−→(n→∞)0,
откуда a0 = b0 = x0 ∈ [a, b]. Пользуясь непрерывностью f на [a, b], имеем:
f(a_n_k) −→(k→∞)f(x0), 
f(b_n_k) −→(k→∞)f(x0).
Так как, по построению, f(ank) > 0, f(bnk

) < 0, то, по теореме о предельном переходе

в неравенствах (13),

(f(x0) ≥ 0) ∧ (f(x0) ≤ 0) ⇒ f(x0) = 0,

и теорема доказана.
